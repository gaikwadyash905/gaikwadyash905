## 🌟 Projects
### [Face Recognition App](https://github.com/gaikwadyash905/Face-Recognition-App)
- **Description**: Designed a Python-based face recognition application capable of identifying individuals in real-time using a webcam feed or input images.
- **Technologies Used**: Python, OpenCV, NumPy
- **Key Features**:
  - Real-time face detection and recognition
  - Generation of YAML and NPY files for training data storage
  - Displaying confidence percentages for identified faces

#### Development Process
1. **Initial Planning**: Structured the project architecture and broke it down into smaller components.
2. **Data Collection and Preprocessing**: Gathered training data from images in local directories and processed it into YAML and NPY files.
3. **Implementation**: Utilized Python-OpenCV and Haarcascade for face detection and recognition, including displaying predicted names and confidence percentages.
4. **Functionality Expansion**: Created separate functions for image and webcam processing, enhancing the app's flexibility and usability.

#### Project Outcome
- Experimented with different Haarcascade face identification XML files to improve accuracy.
- Observed continuous improvement in model accuracy with the addition of more training images.
